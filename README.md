## Introduction
日本の地番住所は、土地の区画を基準に番号が付けられるため構造的に複雑で、初見で場所を推測しにくい特徴がある。一方、韓国の道路名住所は道路を軸とした体系で、道・建物番号により空間把握がしやすいとされ、検索・ナビゲーションとの相性が良いと指摘されてきた。既存の制度比較研究はこうした構造差とその利便性に焦点を当ててきたが、若年層が実際に住所をどの程度使い、制度差をどれほど認識しているかは十分に明らかになっていない。
現代の若年層は地図アプリを前提として行動しており、友人との場所共有はリンク送付、初訪の店は店名検索、必要時だけ住所をコピーして使うなど、住所を中心としない行動が一般化している。制度が異なる日韓であっても、こうした「アプリ依存の位置情報行動」が共通している可能性がある。
そこで本研究では次の仮説を設定する。
若年層にとって住所は「覚える情報」ではなく、必要なときに検索・共有するための“データ”として扱われている。制度差（地番住所・道路名住所）よりも、アプリ利用が行動に強く影響する。
本研究の目的は、日本と韓国の若年層を対象に、住所の記憶・検索・共有に関する実態を比較し、住所制度が日常行動にどの程度影響しているのか、制度差とデジタル行動のどちらが支配的であるのかを明らかにすることである。制度の特徴に触れつつも、若年層の日常行動という新しい視点から住所の役割を再考する。

## Methods
本研究では、若年層の住所利用行動を多角的に把握するため、文献調査とオンラインアンケート調査を組み合わせた。まず文献調査では、日本の地番住所制度の構造的課題（不規則さ・区画基準ゆえの検索のしづらさ）、韓国の道路名住所制度の導入目的（位置把握の容易さ・デジタル地図との整合性）、さらに共同住宅における詳細住所表記の揺れなど、制度そのものの特性を整理した。これにより、制度差がユーザー体験に与える影響を理解する基礎を作った。
次に、18〜29歳の若年層を対象に Google Forms によるアンケートを実施し、計82名（日本57名、韓国25名）から回答を得た。質問項目は、（1）住所の記憶状況、（2）地図アプリの利用頻度、（3）位置情報の伝え方、（4）場面別の住所使用（配達・行政手続き・初訪問など）、（5）住所コピー＆ペーストの行動、（6）住所の必要性に関する意識、（7）住所を直接入力する頻度──といった行動面を中心に構成した。
制度比較の観点を保つため、日本の地番住所の利用経験・検索難易度、韓国の道路名住所の認知・使いやすさに関する設問も含めた。また、地図アプリの種類（Google Map・Naver Map・Kakao Map）や位置共有方法（ピン共有・リンク共有・住所送付）を尋ね、制度差よりアプリの利用文化が行動を規定している可能性を検証できるよう設計した。
データは単純集計とカテゴリ別の比較を行い、国別差・制度差・アプリ利用差をクロスして分析した。
